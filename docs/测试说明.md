# 测试说明

## 依赖

- 开发依赖含 `pytest`：`uv sync --group dev`
- 集成测试需要：本机 **PostgreSQL**、本机 **Milvus**、网络与 **API Key**（嵌入、rerank、LLM）。

## 环境变量

| 变量 | 说明 |
|------|------|
| `DATABASE_URL` | PostgreSQL 连接串，如 `postgresql+psycopg://user:pass@127.0.0.1:5432/dbname` |
| `MILVUS_URI` 或 `MILVUS_HOST` + `MILVUS_PORT` | Milvus 连接 |
| `OPENAI_API_KEY` / `SILICONFLOW_API_KEY` 等 | 调用在线接口的 Key（由 `tests/test.env` 提供） |
| `RAG_TEST_PRINT_ANSWERS` | 设为 `1` 并配合 `pytest -s`，在终端打印 SSE 聚合后的回答摘要 |

## 命令

```bash
cd /path/to/rag_system_demo
uv sync --group dev

# 仅本地、不连外网（健康检查等）
uv run pytest tests/ -m "not integration" -v

# 完整集成（需 Key；建议 source tests/test.env）
set -a
source tests/test.env
set +a
RAG_TEST_PRINT_ANSWERS=1 uv run pytest tests/ -v -s --tb=line
```

## 测试数据
